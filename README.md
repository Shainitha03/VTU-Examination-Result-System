# VTU Examination Result System

A simple web-based system to search and display VTU exam results using USN.

## Features
- Search student result by USN  
- PHP + MySQL backend  
- Styled UI  
- Clean homepage  
- Ready for cloud hosting (000webhost / InfinityFree / Render)

## Tech Stack
- HTML, CSS, JavaScript  
- PHP  
- MySQL Database  
- GitHub for version control  

## How to Run
1. Install XAMPP  
2. Move project folder into `htdocs`  
3. Import `vtu_results.sql` into phpMyAdmin
